# effect-note-cli

To install dependencies:

```bash
bun install
```

To Add :

```bash
bun  bun src/index.ts add "Learn Effect with Bun"
```
To List:

```bash
bun  bun src/index.ts list
```
To Remove:

```bash
bun  bun src/index.ts remove 
```

This project was created using `bun init` in bun v1.3.4. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
